# IT-Ticketing-System
This project focuses on a hands-on IT support lab that demonstrates how to use a ticketing system with Spiceworks. It covers the full support workflow, including receiving and logging issues, assigning priorities, troubleshooting problems, and documenting clear, professional resolution notes.

## Objective
To Learn the fundamentals of an IT ticketing system, including:
- How to properly log a ticket  
- How to classify and prioritize issues  
- Ticket statuses and workflow  
- Hands-on experience using Spiceworks Ticketing system in a HelpDesk setting
